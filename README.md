# 🧠 Mastery Roadmap

The learning journey is divided into structured phases.

---

## 🔹 PHASE 1 — Mathematical Foundations

Build the core math required for true ML understanding.

### Topics

**Linear Algebra**
- Vectors and vector spaces  
- Dot product and projections  
- Matrix multiplication  
- Eigenvalues & eigenvectors  
- SVD intuition  

**Calculus**
- Derivatives  
- Partial derivatives  
- Chain rule  
- Gradients  
- Gradient Descent  
- Convex vs Non-Convex optimization  

**Probability & Statistics**
- Random variables  
- Distributions (Normal, Bernoulli, Binomial)  
- Expectation & Variance  
- Bayes Theorem  
- Likelihood  
- Bias–Variance tradeoff  

**Goal:**  
Understand how models work internally — not just how to use them.

---

## 🔹 PHASE 2 — Core Machine Learning

Classical ML algorithms and professional evaluation.

- Linear Regression  
- Multiple & Polynomial Regression  
- Regularization (Ridge, Lasso)  
- Decision Trees  
- Random Forest  
- Logistic Regression  
- k-Nearest Neighbors  
- Gradient Boosting (conceptual understanding)

### Model Evaluation
- Confusion Matrix  
- Precision / Recall / F1  
- ROC Curve  
- AUC  
- Cross-validation  
- Overfitting vs Underfitting  

**Goal:**  
Build a strong classical ML foundation before deep learning.

---

## 🔹 PHASE 3 — Neural Networks

- Perceptron (mathematical understanding)  
- Activation functions (ReLU, Sigmoid, Tanh)  
- Forward propagation  
- Backpropagation (chain rule derivation)  
- Loss functions (MSE, Cross-Entropy)  

Mandatory:
- Implement a neural network from scratch using NumPy.

**Goal:**  
Understand learning dynamics at equation level.

---

## 🔹 PHASE 4 — Deep Learning & Modern Architectures

- PyTorch / TensorFlow  
- CNNs  
- RNNs (LSTM, GRU)  
- Transformers  
- Attention mechanism  

**Goal:**  
Understand modern AI architectures and representation learning.

---

## 🔹 PHASE 5 — System-Level AI Engineering

- Training pipelines  
- GPU computation basics  
- Batching  
- Distributed training  
- Model checkpoints  
- Model deployment  
- Inference optimization  
- Production ML systems  

**Goal:**  
Move from ML learner → AI engineer.

---

## 🔹 PHASE 6 — Research-Level Thinking

- Read research papers (arXiv, NeurIPS, ICML)  
- Analyze experimental design  
- Reproduce selected research  
- Implement improvements  

**Goal:**  
Think like a researcher, not just a practitioner.

---

## 🔹 PHASE 7 — Mathematical Maturity Upgrade

Advanced study:

- Information Theory  
- Optimization Theory  
- Graph Theory  
- Statistical Learning Theory  

**Goal:**  
Understand why models generalize and where limits exist.
